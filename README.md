# Love Runner

Mini game τύπου offline dinosaur, αλλά σε ρομαντική έκδοση.

## Τι κάνει
- Μία μικρή πίστα
- Το κορίτσι τρέχει προς το αγόρι
- Υπάρχουν λίγα εμπόδια για άλμα
- Όταν τερματίσει εμφανίζεται το μήνυμα: **Σ’ αγαπώ**
- Υποστηρίζει δικές σας φωτογραφίες μόνο στα κεφάλια των χαρακτήρων

## Τεχνολογίες
- Backend: Go
- Frontend: HTML, CSS, JavaScript

## Πώς να τρέξει
```bash
go run main.go
```
Μετά άνοιξε:
```text
http://localhost:8080
```

## Πού βάζεις τα πρόσωπα
Βάλε αρχεία PNG εδώ:
- `static/assets/girl-head.png`
- `static/assets/boy-head.png`

## Ιδέες για επόμενο βήμα
- Να βάλουμε intro με τα ονόματά σας
- Να βάλουμε μουσική
- Να βάλουμε μήνυμα στο τέλος με animation
- Να αλλάξουμε τα εμπόδια σε πράγματα που σας θυμίζουν στιγμές σας
